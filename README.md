# DataVerse 🌌

**DataVerse** is a premium, cinematic, and immersive web experience created for a university AI and Data Science club. Inspired by the visual majesty of *Interstellar* and the minimalist design philosophy of Apple, this project features a physically coherent gravitational singularity as its centerpiece.

## 🚀 Experience the Singularity

DataVerse is not just a website; it's a visual journey. As you descend through the sections, spacetime itself bends around a massive, intelligent black hole.

### ✨ Key Visual Features

- **Relativistic Black Hole**: A physically inspired gravitational singularity with vertex-based gravitational lensing.
- **Dynamic Accretion Disk**: Thousands of GPU-accelerated particles following time-dilated orbital paths.
- **Photon Sphere**: A non-uniform, glowing ring of light that pulses with the tension of immense gravity.
- **Atmospheric Depth**: Layered cinematic haze, starfield lensing, and volumetric light scattering.
- **Apple-Level Minimalism**: A clean, editorial layout with premium typography and smooth transitions.
- **Interactive Spacetime**: Cursor-based gravitational ripples and scroll-driven transformations.

## 🛠️ Technology Stack

- **Framework**: [React](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **3D Engine**: [Three.js](https://threejs.org/) via [@react-three/fiber](https://github.com/pmndrs/react-three-fiber)
- **Shaders**: Custom GLSL for Gravitational Lensing, Doppler Beaming, and Atmospheric Haze.
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Styling**: Tailwind CSS v4

## 📦 Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Rengoku9000/Dataverse.git
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```

## 🎨 Design Philosophy

The core goal of DataVerse is to evoke a sense of awe and "emotional weight." We moved away from generic particle systems toward a "calm, godlike, and mysterious" singularity that feels both ancient and intelligent.
